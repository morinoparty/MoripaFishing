# 🎣 FishingManagerについて

FishingManagerは、釣りに関して管理するクラスです。

---

## 主な機能

魚が食いつくまでの時間を管理します。[LureTimeManager](./lure_time_manager.mdx)


## 設定場所

- フィッシングワールドのポイントごとに設定(ex: つりとぴあのx: 100 y: 100 z: 100 でradius: 50)
- フィッシングワールドごとに設定(ex: つりとぴあ)
- プレイヤーごとに設定
- 釣竿ごとに設定(ex: マングースのマングースフィッシング竿)
- 天気ごとに設定(ex: 晴れ、雨、雷、霧)
- 時間ごとに設定(ex: 1-23, 60-96)
