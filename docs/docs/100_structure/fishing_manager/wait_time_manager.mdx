# ⏳ WaitTimeManagerについて

WaitTimeManagerは、プレイヤーが釣竿を投げてから魚を引くまでの速度を管理するクラスです。

---

## どのように変化させられるか

- 全体として設定(フィッシングワールドごとに設定をオーバーライド)
- フィッシングワールドごとに設定
- Locationおよびradiusを指定して設定
- 釣竿ごとに設定(浮きなど)
- プレイヤーごとに設定

---

## 設定について

デフォルトの値は1.0となっており、倍率等で1以下が設定された場合は、釣りの速度が遅くなります。

- CONSTANT (定数に固定)
- MULTIPLY (倍率)
- ADD (加算)

MUTIPLYがすべて適応された後、ADDが適応されます。 CONSTANTの最大値と比較して大きいほうが適応されます。
