# コンフィグ

## Format

```json title="plugins/MoripaFishing/config.yml"
{
    "database": {
        "type": "SQLITE",
        "host": "localhost",
        "port": 3306,
        "database": "moripafishing",
        "username": "root",
        "password": "password"
    },
    "world": {
        "defaultId": "default",
        "spawnConfig": {
            "spawnMobs": false,
            "spawnMonsters": false,
            "spawnAnimals": false,
            "receiveDamage": false
        },
        "defaultWorldSize": 100.0,
        "defaultClimateConfig":"省略",
        "refreshInterval": 60,
        "defaultTimeZone": "Asia/Tokyo"
    },
    "fishing": "省略",
    "spawn": {
        "spawnMobs": false,
        "spawnMonsters": false,
        "spawnAnimals": false,
        "receiveDamage": false
    },
    "defaultLocale": "ja-JP"
}
```

defaultClimateConfigについては項目が多いため[climate.mdx](./climate.mdx)を参照

fishingについては項目が多いため[fishing.mdx](./fishing.mdx)を参照
