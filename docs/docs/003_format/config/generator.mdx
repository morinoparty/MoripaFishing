# 🌍 ワールドジェネレーターの指定方法

FishingWorldの生成時に、カスタムワールドジェネレーター（例: void, Terra など）を指定できるようになりました。

## 概要

MoripaFishingでは、釣り専用ワールドを作成する際に、バニラのワールドだけでなく、VoidやTerraなどのカスタムジェネレーターを利用することができます。

## サポートされるジェネレーター例
- **Vanilla**（デフォルト）
- **Void**（何も生成しない空のワールド）
- **Terra**（Terraプラグインによる地形生成）
- その他、プラグインで追加されたジェネレーター

## 設定例

```json title="plugins/MoripaFishing/generator/terra.json"
{
  "id": "terra",
  "generator": "Terra:OVERWORLD",
  "type": null,
  "biomeProvider": null,
  "generatorSetting" : null
}
```

## Preset

デフォルトでは、void, terra, normalの3つのジェネレーターが用意されています。

## 注意点
- 指定したジェネレーターがサーバーにインストールされていない場合、ワールド生成に失敗します。
- プラグインベースのジェネレーターを利用する場合は、事前に該当プラグインを導入してください。
- generator名は大文字・小文字を区別する場合があります。

## 関連Issue
- [#44 Support for custom world generators (void, terra, etc.) in FishingWorld creation](https://github.com/morinoparty/MoripaFishing/issues/44)

---

この機能により、より柔軟なワールド構成が可能になります。サーバーの用途や雰囲気に合わせて最適なジェネレーターを選択してください。
