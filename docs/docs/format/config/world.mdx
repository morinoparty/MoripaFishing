# ワールド

## Format

```json title="plugins/MoripaFishing/world/default.json"
{
    "id" : "default",
    "name" : "つりとぴあ",
    "borderSize" : null, //半径
    "borderCentral" : {
        "first" : 0.0,
        "second" : 0.0
    },
    "spawnLocationData" : {
        "x" : 0.0,
        "y" : 64.0,
        "z" : 0.0,
        "yaw" : 90.0,
        "pitch" : 0.0
    },
    "weatherConfig" : null //nullの場合はデフォルトの設定を使用
}

```

## デフォルトの設定

climateConfigの設定は[climate.mdx](./climate.mdx)を参照
