# ディレクトリ

Moripa Fishingは以下のディレクトリ配置をサポートしています。

## ディレクトリ

```bash
plugins/moripa_fishing/
├── config.json
├── world.json
├── database.db
├── lang/
│   ├── en_US.properties
│   └── ja_JP.properties
├── fish/ ここに雑に入れると自動で移動します。
│    └── <rarity_name>/
│        └── <fish_name>.json
└── rarity/
     └── <rarity_name>.json
```
