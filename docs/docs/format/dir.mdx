# ディレクトリ

Moripa Fishingは以下のディレクトリ配置をサポートしています。

## ディレクトリ

```bash
plugins/moripa_fishing/
├── config.json
├── database.db
├── lang/
│   ├── en_US.properties
│   └── ja_JP.properties
├── fish/ 
│    ├── <fish_name>.json #ここに雑に入れると自動で<rarity_name>ディレクトリに移動します。
│    └── <rarity_name>/
│        └── <fish_name>.json
├── world/
│    └── <world_name>.json
└── rarity/
     └── <rarity_name>.json
```
