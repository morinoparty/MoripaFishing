## プリセット一覧

### `beginner`
- 待ち時間倍率: 1.2倍（20%遅い）
- 初心者向けの基本釣竿

### `master`
- 待ち時間倍率: 0.6倍（40%短縮）
- 追加効果: -3秒
- 天候無効化
- 経験豊富な釣り人向け

### `legendary`
- 待ち時間倍率: 0.4倍（60%短縮）
- 固定釣り時間: 2秒
- バイオームボーナス対応

### `speedster`
- 待ち時間倍率: 0.3倍（70%短縮）
- 追加効果: -5秒
- スピード釣り競技用
