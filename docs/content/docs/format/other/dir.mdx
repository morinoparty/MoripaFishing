# ディレクトリ

MoripaFishingは以下のディレクトリ配置をサポートしています。

## ディレクトリ

```bash
plugins/MoripaFishing/
├── config.json
├── moripa_fishing.db
├── translation/
│   ├── en_US.properties
│   └── ja_JP.properties
├── fish/
│    ├── <fish_name>.json
│    └── <rarity_name>/
│        └── <fish_name>.json
├── world/
│    └── <world_name>.json
├── rarity/
│    └── <rarity_name>.json
└── generator/
     └── <generator_name>.json
```
