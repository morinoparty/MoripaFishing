# Git のルール

## Repository

- `https://github.com/morinoparty/MoripaFishing`

## コミットメッセージ

- コミットメッセージは英語で書いてください。
- 形式は `emoji Summary` とします。

例:

```text
🎣 Add weather provider registry tests
```

- 利用する絵文字や傾向は `changelog.config.js` を参考にしてください。

## Issue について

- 新しい機能や大きな振る舞い変更では、必要に応じて Issue を確認または作成してください。
- Issue は英語で記述してください。
- 既存ラベルを優先し、存在しないラベルを勝手に増やさないでください。
